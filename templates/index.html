CROSS SITE SCRIPTING(XSS)

Cross-site scripting (XSS) is a web security vulnerability that allows an attacker to compromise the interactions that users have with a vulnerable application.

SQL INJECTION

SQL injection (SQLi) is a web security vulnerability that allows an attacker to interfere with the queries that an application makes to its database.

COMMAND INJECTION

Command injection which is a cyber attack that involves executing arbitrary commands on a host operating system (OS).

Vulnera a vulnerable application used to help students and anyone keen to know about the vulnerabilities to learn and also get a first hand experience as to how the vulnerabilities work and the individual could try hacking them.

With the help of Vulnera, Students could learn about the latest vulnerabilities by hacking and also understanding them using the LMS while the teachers could make use of this in the class closed environment for any type of experimentations.

As now Vulnera has three rooms for three highly famous vulnerabilities in the information security industry.

Students and Teachers can get hands on experince on vulnerabilities named, Cross Site Scripting(XSS), SQL Injection and Command Injection.

Vulnera was developed by Ravindu Balasooriya and Mark Hettiarachchi, two undergraduates at Sri Lanka Institute Of Information Technology in Malabe Sri Lanka.

Vulnera was developed for a project during year 3 semester 2 in the module Information Security Project in 2022.
